# VERTEX

    Name            Type       Bytes
    -----------------------------------------------------
    pos             vector3    12  
    normal          vector3    12    
    uv              float[2]    8

- `pos` ([vector3](types.md#vector3)) - position in local coordinates
- `normal` ([vector3](types.md#vector3))
- `uv` ([float](types.md)[2]) - texture coordinates, 0;0 is top left, 1;1 bottom right
