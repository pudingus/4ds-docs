# BILLBOARD

    Name            Type       Bytes
    -----------------------------------------------------
    rotationAxis    uint32     4       
    rotationMode    uint8      1       


- `rotationAxis` ([uint32](types.md)) - 0 - x, 1 - y, 2 - z
- `rotationMode` ([uint8](types.md)) - 0 - all axes, 1 - use selected axis
